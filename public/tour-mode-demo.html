üìö Integration Guide

1. Add to your HTML page:

<!-- In your <head> --> <link rel="stylesheet" href="/assets/tour-mode.css"> <!-- Optional: Define custom tour steps (before loading script) --> <script> window.TOUR_STEPS = [ { sel: '#startButton', text: 'Tap "Start Report" to begin.', dwell: 1400, radius: 150 }, { sel: '#w3wField', text: 'Location auto-captured.', dwell: 1600, radius: 160 } ]; </script> <!-- Load tour script with defer --> <script defer src="/assets/tour-mode.js"></script>

2. Add a "Watch Demo" button:

<button id="playDemo" onclick="activateDemoMode()">Watch Demo</button> <script> function activateDemoMode() { // Add ?demo=1 to URL const url = new URL(window.location); url.searchParams.set('demo', '1'); window.location.href = url.toString(); } </script>

3. Theming (edit CSS variables):

:root { --tour-overlay-bg: rgba(0, 0, 0, 0.55); --tour-caption-bg: #ffffff; --tour-caption-fg: #111111; --tour-shadow: 0 10px 30px rgba(0, 0, 0, 0.2); --tour-font: system-ui, sans-serif; --tour-radius: 12px; --tour-cursor-color: #0E7490; }

4. Tour Step Properties:

sel (required): CSS selector for target element

text (required): Caption text to display

dwell (required): Time in milliseconds to show this step

radius (optional): Spotlight radius in pixels (default: 150)
